📏 Usage rules

  • Book only slots that you really intend to use.
  • If you realize you are not going to use a slot, delete the booking again.
  • There is no limit on booking time — try to split the time on shared setups fairly, to the best of your knowledge and abilities.
  • Help others and yourself to get the best science results. 🔬
  • The best setup is a setup that is used — don't be too cautious in booking time if you see there are slots available.

The setups offered in the booking dropdown. Removing one keeps all its existing bookings — it only disappears for future bookings.

    Booking access

    To create or delete bookings, this page needs a GitHub token with write access to this repository. It is stored only in this browser.

    1. Open github.com → Settings → Fine-grained tokens
    2. Resource owner: the repo owner · Repository access: only this repo
    3. Permissions: Contents → Read and write
    4. Generate, copy, and paste it below

    Sharing with the group: one person can create the token and send the others a share link (via private chat — never post it publicly). Opening the link sets the token up automatically.
